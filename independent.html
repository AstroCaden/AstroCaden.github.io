Rotating Neutron Star Modelling

Overview

I began this independent reseach project following my internship at the IFJ PAN in Krakow. It aims to investigate rotating neutron stars using two numerical relativity frameworks: RNS and LORENE, varying the properties, such as: mass, radius, frequency, and central density and analysing the results.

Methods

Creating a C++ program that varies the mass and radius of the neutron star, between set limits, using a predetermined equation of state (EoS), in both LORENE and RNS and exporting each result to a seperate data file. A python program can then be developed and used to take this modelled data and visualise this using both 2 dimensional and 3 dimensinoal plots.

Results

The six-panel figure summarises 2D parameter relationships extracted from the RNS output. Clear correlations appear between mass, radius, and central density, demonstrating the expected relativistic behaviour of compact stars under rapid rotation.

The 3D projection highlights how increasing rotation frequency leads to larger equatorial radii and reduced central densities at fixed mass, consistent with centrifugal flattening.

Summary

This project demonstrates a fully automated workflow for relativistic stellar modelling using RNS and LORENE, combining compiled simulation pipelines in C++ with Python-based post-processing. The resulting visualisations provide an effective framework for exploring how stellar rotation influences equilibrium structure and form a foundation for future work including equation of state variations and gravitational-wave–relevant properties.
